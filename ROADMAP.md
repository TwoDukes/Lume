# Lume — Roadmap

## Planned

- [ ] CORS: restrict to specific origins instead of `*`
- [ ] Input validation on API endpoints
- [ ] Mobile layout optimization
- [ ] Default welcome/onboarding canvas when empty
- [ ] Multi-user support (different dashboards per user)
- [ ] PWA wrapper for mobile

## Ideas

- Persistent canvas URLs (shareable research outputs with custom domains)
- WebRTC for real-time voice interaction
- Canvas templates / starter layouts
- Export canvas to PDF/PNG

## Done

- [x] Password-based browser login with HMAC-signed session cookies
- [x] Canvas snapshot history with TTL, pinning, and privacy controls
- [x] Public share pages (no auth, read-only)
- [x] Progressive block rendering (append blocks in real-time)
- [x] Rate limiting on API endpoints
- [x] Toast notification system
- [x] Full block type support: markdown, code, chart, table, math, mermaid, collapsible, iframe, image
